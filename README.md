# 📇 Business Card

Interaktivní digitální vizitka vytvořená jako originální vlastní projekt. Slouží jako rychlý rozcestník a vizuální vizitka s odkazem na portfolio.

## 🚀 Live Demo
Projekt je nasazený a dostupný online:
👉 [Zobrazit živou ukázku](https://business-card-theta-three.vercel.app)

## 🛠️ Použité technologie
* **HTML5** (sémantická struktura)
* **CSS3** (moderní stylování, vlastní animace a světelné efekty orbu)

## ✨ Vlastnosti projektu
* Plně klikatelná plocha vizitky směřující na portfolio.
* Vizuální efekt zářícího orbu (`glow-orb`) řešený čistě pomocí CSS.
* Responzivní a čistý design postavený od základu bez šablon.
